# Bird in the Forest

A simple browser game: fly a bird through a scrolling forest using the arrow keys.

## How to run

1. Open `index.html` in your browser (double-click the file, or right-click → Open with → your browser).
2. Use **arrow keys** to move: ↑ ↓ ← →

## How to record a clip

- **Windows**: Press `Win + G` to open Xbox Game Bar, then click the record button (or `Win + Alt + R` to start/stop recording). Record the browser window.
- **Other**: Use OBS, ShareX, or any screen recorder focused on the browser tab with the game.

The game runs at 60 FPS; record for a few seconds to get a short clip.
