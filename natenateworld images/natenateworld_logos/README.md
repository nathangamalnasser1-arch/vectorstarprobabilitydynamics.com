## SVG files

SVG files are infinitely scalable — open in any browser, Figma, Illustrator, or Inkscape.
To export as PNG at any size: open the SVG in a browser, right-click → Save as image,
or use Inkscape: inkscape --export-png=output.png --export-width=1920 input.svg

## Usage

- Dark backgrounds: use primary, icon, stream overlay, or stacked
- Light backgrounds / print / merch on white fabric: use natenateworld_light.svg
- Profile pictures / avatars: use natenateworld_icon.svg
- Twitch overlays: use natenateworld_stream_overlay.svg
